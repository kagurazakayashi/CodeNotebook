# **7-Zip GUI → 7z.exe 命令行参数对照表**

## **1. 基础设置**

| GUI 字段   | 示例值           | 命令行参数                 | 说明               |
| -------- | ------------- | --------------------- | ---------------- |
| **压缩包**  | `ICACLS-B.7z` | `7z a "ICACLS-B.7z"`  | `a` = 添加到压缩包     |
| **压缩格式** | 7z            | `-t7z`                | 指定归档格式           |
| **更新方式** | 添加并替换文件       | 默认即可 / `-uq0` 等（下表详述） | 见后面「更新方式对应」      |
| **路径模式** | 相对路径          | 默认                    | 若选“无路径”需 `-spf-` |

---

# **2. 压缩算法相关**

| GUI 字段      | 值        | 命令行参数               | 说明                 |
| ----------- | -------- | ------------------- | ------------------ |
| **压缩等级**    | 9 - 极限压缩 | `-mx=9`             | 0–9                |
| **压缩方法**    | LZMA2    | `-m0=lzma2`         | 选择算法               |
| **字典大小**    | 1024 MB  | `-m0=lzma2:d=1024m` | MB → `m`           |
| **单词大小**    | 273      | `-m0=lzma2:fb=273`  | fb = fast bytes    |
| **固实块大小**   | 64 GB    | `-ms=64g`           | 固实压缩               |
| **CPU 线程数** | 4        | `-mmt=4`            | 多线程数               |
| **压缩共享内存**  | 约 80%    | 无 CLI 参数            | GUI 自动管理，命令行中无对应参数 |
| **分卷大小**    | 若填写      | `-v<size>`          | 如 `-v1g`           |

---

# **3. 选项（Options）**

| GUI 勾选框    | CLI 参数  | 功能            |
| ---------- | ------- | ------------- |
| 创建自解压（SFX） | `-sfx`  | 输出 `.exe` 自解压 |
| 压缩共享文件     | `-ssw`  | 允许压缩被占用文件     |
| 操作完成后删除源文件 | `-sdel` | 压完删源文件        |

---

# **4. 加密（Encryption）**

| GUI 字段   | 示例值      | 命令行参数                                | 说明                     |
| -------- | -------- | ------------------------------------ | ---------------------- |
| **密码**   | XXXXXX   | `-pXXXXXX`                           | 设置密码                   |
| **加密方式** | AES-256  | `-mhe=on`（加密文件名）<br>`-m0=lzma2` 本身即可 | 若勾选“加密文件名” → `-mhe=on` |
| **显示密码** | 无 CLI 参数 | GUI 行为                               |                        |

---

# **5. 时间戳（Options → 时间）**

| GUI 勾选框        | 命令行参数     | 说明                                                    |
| -------------- | --------- | ----------------------------------------------------- |
| 存储修改时间         | 默认        | GUI 默认存储，无需参数                                         |
| 存储创建时间         | `-mtc=on` | ctime                                                 |
| 存储最后访问时间       | `-mta=on` | atime                                                 |
| 设置压缩包时间为当前文件时间 | `-mtt=on` |                                                       |
| 不更新归档文件访问时间    | `-stl`    | store symbolic link?（实际为 "preserve last access time"） |

---

# **6. 更新方式（GUI → 命令行）对照**

| GUI 文本  | 命令行参数       | 意义                                    |
| ------- | ----------- | ------------------------------------- |
| 添加并替换文件 | *(默认，无需参数)* | 已存在则替换                                |
| 添加并更新文件 | `-uq1`      | 仅比源旧的文件才更新                            |
| 完全更新    | `-uq0`      | 总是覆盖                                  |
| 同步更新    | `-uy`       | “同步”语义：更新并删除源中不存在的文件（类似 rsync delete） |
| 跳过已存在文件 | `-aos`      | existing → skip                       |
| 覆盖所有文件  | `-aoa`      | always overwrite                      |
| 仅更新已有文件 | `-u-`       | 不添加新文件                                |

---

# **组合后的完整命令行示例**

假设你当前设置：

* 7z 格式
* LZMA2、字典 1024MB、fb=273
* 固实 64GB
* 线程数 4
* AES-256 + 加密文件名
* 不修改访问时间

则对应命令：

```bash
7z a "ICACLS-B.7z" -t7z -mx=9 -m0=lzma2 -m0=lzma2:d=1024m:fb=273 -ms=64g -mmt=4 -pYOURPASSWORD -mhe=on -stl
```
