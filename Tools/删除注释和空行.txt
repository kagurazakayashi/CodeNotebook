常用删除注释和空行
\/\*[\s\S]*\*\/|\/\/.*
/\*(.|\r\n|\n)*?\*/
^\s*(?=\r?$)\n

---
正则表达式替换，以下替换为 \n


// 单行注释

//[\s\S]*?\n

// 排除对网址的误伤：

//(?!.*\..*\.).*\n


/* 块注释（多行注释） */

/\*(.|\r\n|\n)*?\*/


所有注释

\/\*[\s\S]*\*\/|\/\/.*


html 注释

<!--(.*?)-->


html 注释（有换行）

<!--([\s\S|\r]*?)-->


# 注释

#[\s\S]*?\n
 #[\s\S]*?\n


清除空行，包括空格行和空缩进行

^\s*(?=\r?$)\n


删除双回车

\n\n  ->  \n
