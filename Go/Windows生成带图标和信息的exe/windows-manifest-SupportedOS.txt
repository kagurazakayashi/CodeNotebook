您可以在 Windows 7 或 Windows 8. x 上進行測試，方法是執行資源監視器 (resmon) ，然後前往 [CPU] 索引標籤，以滑鼠右鍵按一下資料行標籤 [選取資料行]，然後核取 [作業系統內容]。 在 Windows 8. x 上，您也可以在 [工作管理員 (taskmgr) 中找到此資料行。 資料行的內容會顯示找到的最大值或 "Windows Vista" 作為預設值。
