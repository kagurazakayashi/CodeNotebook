Ctrl+A + \：退出整个 screen 会话并关闭所有窗口。

Ctrl+A + K：关闭当前窗口，但保留会话和其他窗口。

Ctrl+A + D：这将使你从当前的 screen 会话“分离”（detach）。
会话将继续在后台运行，你可以稍后通过重新连接来恢复它。使用命令：
screen -r

基本操作

Ctrl+A + C：创建一个新窗口（创建一个新的虚拟终端）。

Ctrl+A + N：切换到下一个窗口。

Ctrl+A + P：切换到上一个窗口。

Ctrl+A + 0-9：直接切换到指定编号的窗口。

Ctrl+A + "：列出所有窗口并选择切换。

Ctrl+A + D：分离（detach）当前会话，返回到原始终端。会话在后台继续运行。

会话管理

Ctrl+A + \：退出整个 screen 会话，关闭所有窗口（强制退出）。

Ctrl+A + K：关闭当前窗口（询问确认，如果只有一个窗口，则会退出整个会话）。

Ctrl+A + Q：恢复所有分离的窗口（在多个窗口模式下使用）。

Ctrl+A + X：锁定当前窗口（要求输入密码才能解锁）。

窗口切换和管理

Ctrl+A + W：显示当前窗口的列表。

Ctrl+A + F：在当前窗口中启动搜索。

Ctrl+A + :：进入命令模式，允许输入命令（例如切换窗口、查看窗口列表等）。

Ctrl+A + S：分割当前窗口，显示多个面板。

Ctrl+A + Tab：在分割的窗口之间切换。

日志和输出

Ctrl+A + H：开始/停止当前窗口的日志记录（将所有输出写入文件）。

Ctrl+A + M：查看当前窗口的内存使用情况。

其他功能

Ctrl+A + T：显示当前时间。

Ctrl+A + Z：切换窗口的最大化/恢复。

Ctrl+A + L：恢复窗口的屏幕显示（如果屏幕暂时失去显示）。

退出和终止

Ctrl+A + X：锁定屏幕，需要输入密码解锁。

Ctrl+A + Q：解除所有窗口的锁定。
